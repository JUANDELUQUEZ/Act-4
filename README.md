# API de Gestión de Productos - Actividad 4

Aplicación web backend para la gestión de productos, integrando un sistema de autenticación basada en JWT, operaciones CRUD protegidas y un pipeline de despliegue continuo.

## Tecnologías Utilizadas (Tech Stack)

* **Entorno de ejecución:** Node.js
* **Framework:** Express.js
* **Base de Datos:** MongoDB Atlas (NoSQL)
* **ODM:** Mongoose
* **Seguridad:** JSON Web Tokens (JWT) y bcryptjs
* **Testing:** Jest y Supertest (Apoyado por IA)
* **CI/CD:** GitHub Actions
* **Despliegue (Hosting):** Vercel

## Requisitos Previos

Para ejecutar este proyecto en un entorno local, asegúrese de tener instalado:
* [Node.js](https://nodejs.org/) (v18 o superior)
* Una cuenta y clúster activo en [MongoDB Atlas](https://www.mongodb.com/atlas)
* Git
